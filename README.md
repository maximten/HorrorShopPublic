# Horror Burger Shop

## Setup

Run the Setup Wizard to apply all recommended URP settings:

```
Tools/Horror Burger Shop/Setup Wizard
```

The wizard applies the following settings:

- **Decal Renderer Feature** — adds DecalRendererFeature to the active URP renderer
- **Adaptive Probe Volumes** — enables APV on the URP pipeline asset
- **AO Method** — sets Ambient Occlusion method to Interleaved Gradient
